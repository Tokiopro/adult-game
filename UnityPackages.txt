## 推奨アセット (Asset Store - 無料)
1. Free Classroom Pack - 教室環境
2. School Supplies - 学校小物
3. Anime Character Model - キャラクターベース
4. Japanese School Textures - テクスチャ
5. DOTween (HOTween v2) - アニメーション補助

## インストール手順
1. Unity Editor を開く
2. Window > Package Manager
3. 左上の "+" ボタン > "Add package from git URL..."
4. 各パッケージ名を入力してインストール

## 注意事項
- Universal RPを最初にインストール
- プロジェクト設定でURPアセットを設定
- Input Systemインストール後は再起動が必要
